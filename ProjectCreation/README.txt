Mouse Controls:
===============
Mouse Left/Right -> Look Left/Right
Mouse Up/Down -> Look Up/Down
Mouse Left Click -> Move Forward

Keyboard Controls:
==================
Press 'Esc' -> Pause Game
Hold 'A' -> Red Orb Speedboost
Hold 'S' -> Blue Orb Speedboost
Hold 'D' -> Green Orb Speedboost
Press 'Alt' + 'Enter' -> Fullscreen Mode

Keyboard Controls [CHEATS]:
===========================
Press '1' -> Collect Red Orb
Press '2' -> Collect Blue Orb
Press '3' -> Collect Green Orb
Press 'N' -> Trigger Scene Transition
Press 'Tab' -> Enter Debug Mode
Hold 'Shift' -> Show Wireframe
